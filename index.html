• Result field - current expression and result are displayed here.
  • History field - click on expression to put it in the calculator.
  • Keyboard - you can enter expressions using your keyboard.
  • МR - gets value from memory to current number.
  • MC - clears memory.
  • M+ - set memory value to current memory + current number in calculator.
  • M- - set memory value to current memory - current number in calculator.
  • !x - factorial of x.
  • % - return value of x / 100.
  • x^n - power of n.
  • log2 - log of 2.
  • ln - log of 10.
  • CE - clear expression entry.
  • C - clear entire data.
  • From\To - click and choose system and units to convert from and to.
  • Convert - execute convert button.
